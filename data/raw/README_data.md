# Данные проекта

## Выбранный датасет

Для итогового проекта выбран датасет **Individual Household Electric Power Consumption** из UCI Machine Learning Repository.

Источник: https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption

## Какой файл нужен

Для дальнейшей работы нужно скачать исходный файл:

```text
household_power_consumption.txt
```

и положить его в папку:

```text
data/raw/
```

Ожидаемый путь после скачивания:

```text
data/raw/household_power_consumption.txt
```

## Почему датасет не включен в архив

Исходный файл большой, поэтому в стартовом архиве он не хранится. В репозитории можно оставить ссылку на источник данных или ссылку на облачное хранилище с датасетом.

## Рабочий формат после подготовки

На следующем шаге исходные минутные данные будут агрегированы до часовой частоты и сохранены в файл:

```text
data/processed/prepared_time_series.csv
```

Ожидаемый формат подготовленного файла:

```csv
unique_id,ds,y
household_1,2006-12-16 17:00:00,4.223
household_1,2006-12-16 18:00:00,3.632
...
```

где:

- `unique_id` — идентификатор временного ряда;
- `ds` — временная метка;
- `y` — целевая переменная, среднее часовое значение `Global_active_power`.

## Временный тестовый файл

Файл `sample_daily_sales.csv` остается в проекте только для проверки структуры и smoke-тестов. Он не является финальным датасетом для сдачи.
